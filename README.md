# My First Next.js App

This is my first project built using **Next.js**.
The purpose of this project is to learn how to build modern React applications using Next.js features such as routing, optimization, and reusable components.

---

## Description

This project helped me understand:
- How Next.js works on top of React
- Page-based routing
- Component structure
- Basic project organization

---

## Features

- Fast performance with Next.js
- Page-based routing
- Responsive UI
- Reusable components
- SEO-friendly pages

---

## Tech Stack

- **Next.js**
- **React**
- **TypeScript**
- **CSS / Tailwind CSS / SCSS**
